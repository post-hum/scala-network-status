# Scala (XLA) Network Status

Real-time network monitoring for Scala cryptocurrency.

## Live Page

**https://post-hum.github.io/scala-network-status/**

## Features

- Current block height, difficulty, hashrate
- Circulating and total supply
- Historical difficulty charts
- Historical hashrate charts
- Auto-updates every 10 minutes

## Technical

- GitHub Actions for data collection (every 10 min)
- GitHub Pages for hosting
- Chart.js for graphs
- Pure HTML/CSS/JS, no dependencies
